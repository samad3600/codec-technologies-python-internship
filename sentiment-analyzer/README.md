# ✨ SentiSense AI — AI Based Sentiment Analyzer

SentiSense AI is an AI-powered sentiment analysis web application that detects emotions from text and predicts sentiment with confidence scoring using NLP.

---

## 🚀 Features

- Real-Time Sentiment Analysis
- Positive & Negative Detection
- Confidence Percentage
- Interactive Glassmorphism UI
- Recent Analysis History
- NLP Powered Predictions

---

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

AI:
- Hugging Face Transformers
- NLP
